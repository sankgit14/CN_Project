 CN Project - Client Server Communication

 Overview
This project demonstrates a simple **client-server communication system** using Python.  
It is designed to showcase fundamental concepts of **Computer Networks**, including socket programming and data exchange between systems.

---

 Features
- Establishes connection between client and server
- Sends and receives messages
- Demonstrates basic networking concepts
- Lightweight and easy to understand

---
 Technologies Used
- Python
- Socket Programming
- TCP/IP Protocol

---

  Project Structure

---

 Setup Instructions

 1. Clone the repository

 2. Requirements
- Python 3.x installed  
- No external libraries required  

---

 Usage Instructions

    Step 1: Start the server

    Step 2: Run the client (in a new terminal)

---

 How It Works
- The server starts and listens for incoming connections  
- The client connects to the server  
- Once connected, messages are exchanged between them  

---

 Example Workflow
1. Run the server  
2. Run the client  
3. Establish connection  
4. Exchange messages  

---

 Learning Outcomes

- Hands-on experience with socket programming  
- Basics of TCP/IP communication
